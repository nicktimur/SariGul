# Sarı Gül

Sarı Gül, Unity oyun motoru kullanılarak geliştirilmiş bir 2D aksiyon platform oyunudur. Oyuncular, zorlu platformlar ve düşmanlarla dolu seviyelerde ilerlerken reflekslerini ve stratejik düşünme yeteneklerini test eder.

## Özellikler
- **Platform Mekanikleri**: Hareketli platformlar, asansörler ve zıplama gibi klasik platform öğeleri.
- **Düşmanlarla Mücadele**: Oyuncular, çeşitli saldırı yetenekleriyle düşmanları alt eder.
- **Seviye Tasarımı**: Farklı temalara sahip bölümler ve zorluk seviyeleri.
- **Piksel Sanat Tarzı**: Retro görsellerle zenginleştirilmiş bir oyun deneyimi.
